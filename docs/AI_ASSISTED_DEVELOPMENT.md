# AI-Assisted Development Traceability

## Purpose
This document explains how AI is used responsibly as a copilot.

## Workflow
1. Human defines task and constraints
2. AI suggests options
3. Human reviews and validates
4. Tests are executed
5. Human commits

## Tools
- ChatGPT for planning
- Claude for coding

## Principles
- Human-in-the-loop
- TDD first
- Clean Architecture boundaries respected
